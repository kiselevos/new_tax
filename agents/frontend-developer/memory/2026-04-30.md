# 2026-04-30 - Frontend Developer

## [начало сессии] SEO: бренд в заголовках и JSON-LD Article schema

- Что: добавлен бренд " | Калькулятор НДФЛ" к `<title>` и JSON-LD `Article` schema в `<head>` на пяти info-страницах: `about.tmpl`, `regional_info.tmpl`, `special_tax_modes.tmpl`, `tax_deductions.tmpl`, `employment_types.tmpl`.
- Результат: успех, `go build ./...` прошёл чисто.
- Вывод:
  - JSON-LD вставляется после `<link rel="canonical">` и перед первым `<link rel="stylesheet">` — это устоявшийся порядок в проекте.
  - В `description` поля JSON-LD убирается `{{.CurrentYear}}` — это Go-шаблонное выражение не нужно в статической schema, подставляем реальный текст без него.
  - `headline` для `tax_deductions` содержит тире (—) — нужно использовать em-dash, не дефис, чтобы совпадало с `<title>`.
  - В `special_tax_modes.tmpl` meta description содержит `{{.CurrentYear}}` — в JSON-LD подставляем статический текст без шаблонного выражения, иначе Go-шаблон выведет `{{.CurrentYear}}` как есть в JSON, что неверно семантически.
