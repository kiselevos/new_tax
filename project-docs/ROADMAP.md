## Фаза 3 — Типы занятости: ГПХ (текущий приоритет)

**Зачем:** расширить аудиторию за счёт фрилансеров и подрядчиков. Дать возможность осознанно сравнить условия найма.

### UI — выбор типа занятости
- [ ] Блок «Тип занятости» на главной форме — вне аккордеона, по умолчанию ТД
- [ ] Три взаимоисключающих варианта: **Трудовой договор / ГПХ / Самозанятый**
- [ ] Ссылки на инфо-страницы под каждым вариантом
- [ ] Блок «Особые режимы» (силовики, нерезиденты) переезжает под блок занятости в аккордеон
- [ ] При несовместимых комбинациях — информационное сообщение (не блокировка)

### Расчёт ГПХ
- [ ] Убрать ФСС (2.9%) из взносов работодателя
- [ ] Плашка на странице результата: нет больничных, отпускных, защиты по ТК
- [ ] НДФЛ и остальные взносы — без изменений

### API
- [ ] Добавить `employment_type` enum: `TD` / `GPH` / `SELF_EMPLOYED` в proto
- [ ] Валидация несовместимых комбинаций — строгая: `400 Bad Request` с описанием
- [ ] Обновить документацию API

### Инфо-страницы (параллельно)
- [ ] Авторские гонорары (ст. 221 НК РФ, профвычет 20–40%)
- [ ] Дивиденды (13%/15%, без страховых взносов)
- [ ] ИП на УСН (6% с дохода или 15% с прибыли)
- [ ] ИП на патенте (фиксированная стоимость, региональная специфика)

**Сложность:** низкая для ГПХ, низкая для инфо-страниц.

## Фаза 4 — Самозанятые (НПД)

**Зачем:** самозанятость — самая популярная альтернатива ТД. Своя налоговая логика, своя аудитория.

Ключевые особенности:
- Ставка 4% (доход от физлиц) или 6% (доход от юрлиц)
- Нет обязательных страховых взносов
- Вычет 10 000 ₽ при регистрации (снижает ставку до 3% / 4% до исчерпания)
- Лимит дохода 2 400 000 ₽ в год
- НПД доступен гражданам РФ и стран ЕАЭС; нерезиденты — отдельный кейс

- [ ] Расчёт НПД в рамках единого блока «Тип занятости»
- [ ] Поле: источник дохода (физлицо / юрлицо)
- [ ] Учёт вычета при регистрации (опционально)
- [ ] Предупреждение при превышении лимита 2.4М ₽/год

**Сложность:** средняя — другая налоговая логика, но встраивается в существующий UI.

## Фаза 5 — ИП

**Зачем:** следующий шаг после самозанятости для тех, кто растёт. Поможет принять решение о форме.

Открытые вопросы (решить перед стартом):
- Какие режимы покрываем в первую очередь: УСН «доходы», УСН «доходы минус расходы», патент?
- Региональная специфика патента — насколько глубоко идём?
- Фиксированные взносы ИП — считаем автоматически или вводим вручную?

**Сложность:** средняя-высокая. Стартуем после самозанятых, когда появится понимание аудитории.

## Фаза 6 — Сравнение режимов

**Зачем:** пользователь вводит сумму дохода — видит все режимы рядом. Инструмент принятия решений о форме занятости.

- [ ] Таблица: одна сумма → ТД / ГПХ / Самозанятый / ИП (налог, взносы, на руки, защита)
- [ ] Учёт районного коэффициента и северной надбавки в режиме ТД/ГПХ
- [ ] Подсветка оптимального варианта

**Сложность:** средняя. Реализуем после того, как все режимы посчитаны корректно.

## Фаза 7 — Личный кабинет

**Зачем:** отслеживать налоги в течение года, а не пересчитывать каждый раз.

Открытые вопросы:
- Хранить данные на сервере или достаточно URL с параметрами?
- Минимальный сценарий: «сохранить настройки» или «история по месяцам»?
- Авторизация: Google / Telegram / email?

**Сложность:** высокая. Требует авторизации, хранения данных, отдельного архитектурного решения.
